Shkurtoret

Per ta bere kete nje pervoje me te lehte dhe me te kendshme per ju, sigurisht qe kemi perfshire disa shkurtore.

Ketu jane shkurtoret qe kemi perfshire ne aplikacionin tone:

  • Enter => Kycu / Regjistrohu
  • Alt + H => Shkoni tek faqja kryesore
  • Alt + U => Shkoni tek faqja per te perditesuar punjones
  • Alt + A => Shkoni tek faqja per te shtuar punjones
  • Alt + V => Shkoni tek faqja per te pare punjonesit
  • Alt + X => Mbyllet Aplikacioni
  • Alt + L => DILNI
  • Ctrl + A => Gjuha shqipe
  • Ctrl + E => Gjuha Angleze
  • Ctrl + H => Seksioni i Ndihmes
  • Ctrl + Enter => Shto / Perditeso Punonjesin
  • Ctrl + Delete => Pastro Fushat
