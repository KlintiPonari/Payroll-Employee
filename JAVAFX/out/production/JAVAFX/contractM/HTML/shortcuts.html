ShortCuts

To make this an easier and more pleasant experience for you, of course we have included a few shortcuts.

Here are the shortcuts we've included in our application:

  • Enter => Sign in / Sign Up
  • Alt + H => Navigate to Home Page
  • Alt + U => Navigate to Update Employees Page
  • Alt + A => Navigate to Add Employees Page
  • Alt + V => Navigate to View Employees Page
  • Alt + X => Closes Application
  • Alt + L => LOGOUT
  • Ctrl + A => Albanian Language
  • Ctrl + E => English Language
  • Ctrl + H => Help Section
  • Ctrl + Enter => Add/Update Employee
  • Ctrl + Delete => Clear Fields
